Features

Multiple Visualization Types

Choose from different visualization layouts to best represent your code structure:

  • Force-directed: Nodes repel each other while edges act like springs (default)
  • Hierarchical: Tree-like visualization showing inheritance and containment
  • Radial: Circular layout with most connected nodes in the center
  • Sankey: Flow diagram showing dependencies between components

Interactive Exploration

Interact with your code visualization in various ways:

  • Zoom in/out with the mouse wheel
  • Click and drag nodes to reposition them
  • Hover over nodes to see details
  • Click on nodes to highlight connections and see node details
  • Search for specific nodes using the search box

Node Filtering

Limit what's displayed to focus on what matters:

  • Filter by node type (file, directory, function, class, etc.)
  • Text search to find specific components
  • Control the maximum number of nodes displayed
  • Focus on a specific node and its immediate connections

CLI Commands

Generate a visualization using the CLI:

codestory visualize generate --output visualization.html

Customize your visualization:

codestory visualize generate --type hierarchy --theme dark --output visualization.html

Open an existing visualization:

codestory visualize open visualization.html

List all generated visualizations:

codestory visualize list

Note: This is an example page. The actual visualization generated by the CLI will be interactive and contain your real codebase structure.
